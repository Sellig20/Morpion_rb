Bienvenue dans mon jeu de Morpion !

Langage utilise :

```Ruby 2.7.1```

Petit jeu de morpion simple, 2 joueurs max, choisis un pseudo pour chacun et un des deux symboles (ou "token") suivants : X/O.
Dans le tableau cree, qui comporte 9 cellules, chaque ligne horizontale est representee par les lettres A,B et C, et chaque ligne verticale est representee par les chiffres 1,2 et 3. Le but est de remplace chaque petit point par le token choisi.
En fonction du resultat, match nul ou 1 gagnant.
Possibilite de rejouer.
